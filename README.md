# Ph.D. Positions in the AEGIS-R Lab  
Department of Civil, Structural, and Environmental Engineering  
University at Buffalo (SUNY)

This repository hosts the files for the interview question for the Ph.D. position in the AEGIS-R Lab.

---

## Repository Structure

This README file provides general information about the repository structure.  
The **problem statement** is located in `problem_statement.md`.  
The **Analyses** folder contains all scripts used for preprocessing and calculations.  
The **Data** folder includes all processed data frames produced by the analysis scripts.  
The **Raw_files** folder contains all input data in their original raw form.

```
phd-position-aegis-r/
│
├── README.md
├── problem_statement.md
│
├── Analyses/
│   └── preprocessing/
│       └── compute_ba18_residuals.py
│
├── Data/
│   └── residuals/
│       └── NGA2West_FAS_CA_residual.csv
│
└── Raw_files/
    └── NGA2West_FAS_CA.csv
```
